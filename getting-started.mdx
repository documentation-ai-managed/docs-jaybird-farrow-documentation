## Onboarding Process

The onboarding module introduces the T.E.A.C.H and L.E.A.R.N frameworks through interactive videos and quizzes. You complete a self-assessment to identify knowledge gaps in areas like legal boundaries or GNSS technology. This sets the foundation for personalized learning recommendations.

<Steps>
  <Step title="Verify Email" icon="mail">
    Click the verification link sent to your inbox. This activates full platform access within `<1 minute`.
  </Step>
  <Step title="Set Preferences" icon="sliders">
    Select your region and experience level. Preferences influence module suggestions, such as focusing on U.S. PLSS standards.
  </Step>
  <Step title="Explore Dashboard" icon="layout">
    Navigate the interface to find courses, forums, and progress trackers. Bookmark key sections for quick access.
  </Step>
  <Step title="Join Community" icon="users">
    Participate in your first discussion thread on current surveying trends.
  </Step>
</Steps>

<Callout kind="tip">
  Enable two-factor authentication for added security when accessing sensitive project data.
</Callout>

## Integrating with Tools

Connect Jaybird Farrow to your existing software like AutoCAD or Trimble Business Center. Export learning certificates directly to your resume builder. Participate in virtual field trips to observe advanced surveying in action, enhancing your practical understanding.

<CodeGroup tabs="Bash,Python">
  ```bash
  # Syncing data with external tools
  git clone https://repo.surveytools.com/jaybird-integration.git
  cd jaybird-integration
  pip install -r requirements.txt
  python sync.py --api-key YOUR_KEY --project "field-survey"
  ```
  ```python
  import requests
  
  def sync_certificates(api_key, user_id):
      headers = {'Authorization': f'Bearer {api_key}'}
      response = requests.get(f'/api/certificates/{user_id}', headers=headers)
      if response.status_code == 200:
          return response.json()['certificates']
      return []
  
  certs = sync_certificates('your_api_key', 'user123')
  print(certs)
  ```
</CodeGroup>
